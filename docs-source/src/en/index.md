### Bring it on! Let reflection become poetic and picturesque

```java
public class World {

    private void sayHello(String content) {
        System.out.println("Hello " + content + "!");
    }
}
```

```kotlin
val newWorld = World()
classOf<World>().method {
    name = "sayHello"
    param(StringClass)
    type = UnitType
}.get(newWorld).call("YukiReflection")
```
